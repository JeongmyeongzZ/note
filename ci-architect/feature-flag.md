# feature flag (toggle)

- ON/OFF switch 
- Permissioning Toggle
- Canary Toggle
  - 일부 지역이나, 일부 특정 유저라는 application level 의 더 자유로운 canary
- 장애 대응
- 개발 도중에 언제든지 메인 브랜치에 통합해두거나, 다른 운영 기능들과 함께 운영 서버에 배포하는 것이 자유롭습니다.
